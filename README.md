## ✨ Features

* 🏨 Elegant premium UI (Hotel landing experience)
* 📅 Smooth room booking system
* 📱 Fully Responsive (Mobile + Tablet + Desktop)
* ⚡ Fast performance using Vite
* 🔐 Secure backend APIs

---

## 🛠️ Tech Stack

### 🎨 Frontend

* React.js (Vite)
* Tailwind CSS

### ⚙️ Backend

* Node.js
* Express.js

### 🗄️ Database

* MongoDB Atlas

### 💳 Payments

* Razorpay

### ☁️ Deployment

* Frontend → Vercel
* Backend → Render
